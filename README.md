🎉 Event Management Website
A dynamic, visually rich event management web application built with vanilla HTML, CSS, and JavaScript. The site showcases upcoming shows, featured artists, a media gallery, event experiences, and a blog — all powered by a central data file for easy content management.
🔗 Live Demo: event-management-three-omega.vercel.app

✨ Features

🎟️ Upcoming Shows – Browse and explore scheduled events
🖼️ Gallery – Visual memories from past events
🎤 Featured Artists – Spotlight on performing artists and lineup
🌟 Experiences – What the event offers to attendees
📰 Blog / Articles – Latest news and updates
📱 Responsive Design – Optimized for all screen sizes
⚡ Data-driven – All content managed via a single data.js file

🚀 Getting Started
No build tools required — this is a pure HTML/CSS/JS project.

Clone the repository:

bashgit clone https://github.com/MRaashu76/event-management.git
cd event-management

Open in browser: Open index.html directly, or use the VS Code Live Server extension.


📖 Usage
All site content is managed in data.js. To add or update events, artists, gallery items, or blog posts, edit the relevant arrays/objects there — no changes to HTML or JS needed.

🌐 Deployment
Deployed on Vercel. To deploy your own fork:

Push to GitHub
Import the repo at vercel.com
Vercel auto-detects the static site and deploys instantly
